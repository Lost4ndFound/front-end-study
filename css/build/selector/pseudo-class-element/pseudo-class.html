许我们的心里藏有一个海洋，流出来的却是两行清泪，可我并不感到孤单，我 只是简单地喜欢书写凄凉。以前我们敷衍的情绪，在别人记忆的曲线里渐行渐远渐 无言。难道以前的悸动，只是岁月留给我一个人的错觉，我有点不相信自己的眼睛。 我没有哭泣，只是学会了思念而已。
